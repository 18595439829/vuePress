# 使用vuepress编写文档

# 启动方式

  1. 拉取代码
git clone https://github.com/18595439829/vuePress.git
  1. 运行yarn docs:dev(推荐)或npm run docs:dev
yarn docs:dev // 推荐该方式
// 或
npm run docs:dev

# 使用vue + markdown编写文档

  1. 拉取代码
git clone https://github.com/18595439829/document.git
  1. 切换分支到vuePress
git checkout markdown
  1. 运行yarn serve或npm run serve
yarn serve 
// 或
npm run serve
